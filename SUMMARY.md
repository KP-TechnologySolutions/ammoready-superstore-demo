## 🛠️ Development Workflow

### **Getting Started:**
```bash
cd ammoready-superstore-demo
npm install
npm run build:serve  # ⚠️ Never use npm run dev (RAM constraint)
```

### **Available Commands:**
- `npm run build:serve` - Build and serve (recommended for development)
- `npm run build` - Production build only
- `npm run serve` - Serve existing build
- `npm run clean` - Clean build artifacts

### **⚠️ Critical Constraints:**
- **Never use `npm run dev`** - 8GB RAM constraint requires build+serve workflow
- **Always use static generation** - Performance and reliability priority
- **Professional image sourcing** - Real manufacturer photos only
